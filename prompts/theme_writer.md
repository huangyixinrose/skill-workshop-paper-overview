# theme_writer —— 一条主线的综述（子代理，每类一个）

模板：

---
你在为一份面向 **{AUDIENCE}** 的领域综述撰写其中一条主线。

- 主线名称：{category}
- 本主线相关论文的结构化提要（JSON 数组，可能为空或很少）：
{digests_json}

请用中文写这一节综述（600–1000 字）：
1. 本届这条主线的**核心进展**是什么；
2. **点名代表作**，并串起它们共同的方法取向、以及彼此的分歧/互补；
3. 给 {AUDIENCE} 的**可落地启发**；
4. 若本主线论文很少或为空，简要说明，并指出该方向当前的**留白**。

平实但专业，**不要用"不是 X 而是 Y"的句式**。返回纯文本，可用简单小标题（## / ###）。
---

产出顺序必须与 `cats.json` 对齐，写入 `data/sections.json`（list[str]）。
